# Bookstore React Js
Frontend and backend Book Store Website Application using `React List`, `React Router`, `Local and Session Storage`, `Axios`, `MySQL`, `Express`.

## Feature
1. Dashboard
2. Gallery (CRUD, Add To Cart, Session Storage For Customer)
3. Officer (CRUD)
4. Customer (CRUD)
5. Contact
6. About
